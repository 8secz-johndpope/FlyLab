# Scala 练习

## sdkman

> curl -s "https://get.sdkman.io" | bash

```bash
sdk install scala 版本号
sdk install java 版本号
# 暂时只支持java8
sdk install gradle 版本号
# 初始化项目版本
gradle init
# 编译构建
./gradlew build
# 还有就是配置java仓库

```
