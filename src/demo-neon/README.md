# RUST + NODEJS = NEON

> 用于编写安全、快速的原生 Node.js 模块的 Rust 绑定。
为Nodejs创建更多的应用场景
@see https://guides.neon-bindings.com/
## 示例
Neon 让我们可以在 Rust 代码中使用 JavaScript 类型。要创建一个新的 Neon 工程，我们可以使用它自带的命令行工具。执行 npm install neon-cli --global 来安装它。
